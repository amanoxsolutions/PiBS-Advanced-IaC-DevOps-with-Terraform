Exercise

  1. Create a CI/CD pipeline using Terraform
  1. Create another CI/CD pipeline using CloudFormation
  1. Choose an application that you’ve already developed and deploy it using one of the CI/CD pipelines you’ve created in 1. or 2. Build jobs will be based on CodeBuild. Host code in a git repository. Use CodeDeploy when is possible.

    Target for deployments:

    • On EC2
    • On a serverless platform (ECS/Fargate or Lambda). In case of containers, they have to be hosted in ECR.
    • On your local machine using CodeBuild agent (just the build part is enough)
  1. Prepare a demo to show how to deploy updates to the app live via git push + CI/CD
